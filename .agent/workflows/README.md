# Kịch bản thực thi (`workflows/`)

Thư mục này chứa các kịch bản (workflows) thực thi từng bước (step-by-step) hoặc định nghĩa các slash commands để người dùng kích hoạt Agent thực hiện một chuỗi công việc cụ thể.

## Các ví dụ về kịch bản:
1. **`generate_automation.md`**: Luồng tự động đọc tài liệu mô tả, sinh mã kiểm thử, lưu vào thư mục `src/tests`, và chạy kiểm thử thử nghiệm.
2. **`regression_test_report.md`**: Kịch bản chạy bộ kiểm thử hồi quy, tổng hợp kết quả và xuất báo cáo PDF/Markdown.
