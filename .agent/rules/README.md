# Quy tắc hoạt động (`rules/`)

Thư mục này dùng để lưu trữ các tệp quy tắc (dưới dạng markdown `.md` hoặc định dạng khác) mà AI Agent **bắt buộc** phải tuân theo khi tương tác với dự án này.

## Các ví dụ về quy tắc có thể thêm vào:
1. **`coding_standards.md`**: Quy định về chuẩn viết mã nguồn (TypeScript, Playwright/Cypress, ESLint rules).
2. **`naming_conventions.md`**: Quy định đặt tên file, test suite, test case, locator, variables.
3. **`git_guidelines.md`**: Quy định về commit message, quy trình tạo nhánh (branching strategy).
